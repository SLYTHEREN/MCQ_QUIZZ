The following are the rules for MCQ Round

  • Each Question will be timed. A running timer of 30 seconds is shown while the participant answers the question. If a participant fails to register an answer within the allotted time, they will receive no point and the round continues with next question.
  • Ipsum modi
  • Left to be changed as there are no lifelines
  • The questions currently come from a configuration file. However, there are plans ( we need to work on this as well) to integrate a backend or get the question from some other method.
  • If you would like to feed your own questions and play with your friends, please navigate to Upload Questions screen. The Upload Questions screen has option for either using the system user interface to upload questions or you can upload the json file itself.
  • For any technical problems during the MCQ round it will be managed by the organizing team.
